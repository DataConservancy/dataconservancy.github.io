Data Conservancy Developers Home

This website maintains helpful links to projects and documents created by the Data Conservancy.

BagIt Profile

The Data Conservancy BagIt Profile refines the existing BagIt specification in order to provide platform interoperability and Resource Manifest discovery when combined with the packaging specification.

Packaging Specification

The Data Conservancy Packaging Specification relies on the BagIt Profile to package and describe digital content.

Packaging Tool

The Data Conservancy Packaging Tool is a JavaFX GUI for creating packages that conform to the Data Conservancy Packaging Specification.
